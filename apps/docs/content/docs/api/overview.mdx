# API Overview

## Base URLs

- REST: `https://api.ego.exchange`
- WebSocket: `wss://ws.ego.exchange`

## Authentication

- Wallet signature for user sessions
- API key and scoped secret for bots
- Session keys for delegated agent actions

## Surface areas

- markets
- orderbooks
- trades
- orders
- positions
- balances
- rewards
- staking
- governance
- AI agents
- launches
