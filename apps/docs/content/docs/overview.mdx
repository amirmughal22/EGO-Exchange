# Overview

Ego Exchange is a permissionless cross-chain exchange built for pro traders, autonomous agents, market makers, and new token launches.

## Core capabilities

- Spot, perpetuals, options-ready architecture
- Unified margin system
- Onchain settlement on Lithosphere Makalu
- Cross-chain routing and bridge adapters
- AI copilot, AI agents, alpha feed, meme markets
- Launchpad, liquidity incentives, and governance
